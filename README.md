# Ascension 

Ascension is a small action-platformer that I created during my last game Jam. You play a little blue guy in its climb of a mountain.

Controls
--------
Keyboard:
- W/Z - Jump
- A/Q - Move left
- D - Move right
- Space - Shoot
- R - Reset
- Esc - Pause

Controller:
- DualShock Cross  - Jump
- D-Pad - Move left
- D-Pad - Move right
- DualShock Circle - Shoot
- DualShock Triangle - Reset
- Start - Pause
